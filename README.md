# FLUXDOCTOR v1.0

Apple Disk II diagnostic utility for real-time
troubleshooting, diagnositcs, and repair.

Available as an standalone binary (`FLUXDOCTOR`), or packaged as a 140KB bootable
DOS 3.3 floppy disk image (`FLUXDOCTOR.DO`).

# Prerequisites

## Windows

1. Install **dasm** from https://dasm-assembler.github.io/

2. Install **AppleWin** from https://github.com/AppleWin/AppleWin

3. Install **AppleCommander** from https://applecommander.github.io/ac/
